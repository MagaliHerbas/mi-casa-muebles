Como trabajamos

Proceso para obtener tu mueble a medida

Para que podamos ofrecerte un servicio personalizado y de calidad, te explicamos el proceso a seguir: El primer paso es que el cliente envíe las medidas de los muebles que desea fabricar. Con esta información, nuestro equipo elaborará un presupuesto detallado. Una vez que el cliente apruebe el presupuesto y decida avanzar con nosotros, coordinaremos una visita a su domicilio. Durante esta visita, tomaremos las medidas finales y discutiremos los detalles de diseño y distribución para asegurar que el mueble se ajuste perfectamente a tus necesidades y espacio. Para confirmar el inicio del trabajo, se requiere una seña del 60% del valor del presupuesto. El resto del pago se realizará una vez que el trabajo esté terminado y el cliente esté completamente satisfecho con el resultado.
